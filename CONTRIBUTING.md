# Contributing to BinaryTriage-X

Thank you for your interest in contributing to BinaryTriage-X.

BinaryTriage-X follows the Blackwell Toolworks engineering and documentation standards.

Before contributing, review the complete contribution guide:

- [Development Workflow](docs/14-development-workflow.md)
- [Coding Standards](docs/08-coding-standards.md)
- [Testing Strategy](docs/09-testing-strategy.md)
- [Code Review Checklist](docs/28-code-review-checklist.md)
- [Security Review Checklist](docs/27-security-review-checklist.md)

All changes must include appropriate tests and documentation updates.

Do not submit live malware, unauthorized samples, private customer data, or restricted datasets.
